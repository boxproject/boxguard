boxgd作为签名机(voucher)的伴生程序用来为签名机保驾护航，其使用也非常地简单。


编译：
make build
编译后会生成build


使用方法：
cd /build/bin
1.安装服务

```bash
sudo ./boxgd install
```

2.启动服务：

```bash
sudo ./boxgd start
```



3.停止服务:

```bash
sudo ./boxgd stop
```



4.查看服务状态：

```bash
sudo ./boxgd status
```



5.卸载服务：

```Bash
sudo ./boxgd remove
```
the end
